# SQE1 Materials Comparison

Purpose: decide which materials belong in the study system and what function each material serves.

| Material type | Use | Risk | Repository response |
|---|---|---|---|
| Official SQE pages | Exam format, specification, sample style | May change annually | Source map and date checks |
| SRA Standards and Regulations | Ethics and professional conduct | Must check current version | Ethics overlay in every subject |
| Public legislation | Statutory rules | Amendments and commencement | Source anchor per topic |
| Public procedure rules | Civil and criminal practice | Rule updates | Procedure sequence sheets |
| Generated coursebook | Rule learning | Can become too generic | Audit rule: no filler sections |
| Generated workbook | Application practice | Needs quality control | Error-code feedback loop |
| Issue atlas | Final recall | Can oversimplify | Link every card to a rule and trap |

## Decision

Use official and public sources as anchors. Use generated materials as structured learning aids. Keep an error log so weak answers update the materials.
