# SQE1 30 Day Starter Plan

Target: SQE1 January 2027
Audience: Korean-qualified lawyer preparing in Korean with English legal terminology.

## Daily operating rule

Each study day has four blocks:

1. Rule acquisition: read and summarise rules.
2. Application: solve short client scenarios.
3. Error log: classify wrong answers by rule, fact, exception, procedure, ethics.
4. Recall: close-book rewrite of the day's rule map.

## Error coding

Use these codes in the dashboard:

- R: rule not known.
- F: fact misread.
- E: exception missed.
- P: procedure sequence error.
- ETH: ethics issue missed.
- CALC: calculation or date error.
