# SQE1 Rebuild Audit

Date: 2026-07-04

## Finding

The previous material was structurally large but educationally weak. Page count was created by repeated generic paragraphs rather than real SQE learning content.

## Main defects

| Defect | Why it fails |
|---|---|
| Repeated topic shell | The same explanation appeared under many headings with only the topic name changed. |
| No rule density | Subjects such as directors duties, civil procedure, trusts, and property registration need concrete rules and sequences. |
| No application | SQE1 tests application to client facts, not passive topic descriptions. |
| No traps | Good SQE material must flag exceptions, timing traps, authority issues, ethics traps, and procedural order. |
| No answer-elimination method | SBA questions require selecting the best answer among plausible options. |
| No error feedback loop | Wrong answers must update rule sheets and issue maps. |

## Rebuild criteria

A section is acceptable only if it contains at least four of the following:

1. Legal rule or procedural step.
2. Source anchor.
3. Client-fact trigger.
4. Trap or exception.
5. Ethics angle.
6. Application example.
7. Self-test item.
8. Explanation of why an alternative answer fails.

## Specific correction to earlier bad format

Bad format:

> Topic X is important. Know the trigger words. Apply the statute. Answer the question.

Correct format:

> Directors owe separate statutory duties. A question may test whether the board acted within powers, promoted the success of the company, avoided conflicts, declared an interest, or used corporate property for personal benefit. The trap is that shareholder approval may cure some issues but not all, and insolvency shifts practical focus toward creditors.

## Output strategy

The rebuilt project uses:

- one theory textbook for rule maps and client application;
- one practice workbook for original SBA-style drilling;
- one issue atlas for final revision;
- a dashboard for error-loop operation.
