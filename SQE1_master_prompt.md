# SQE1 Master Prompt

Goal: create an original SQE1 study system for the January 2027 sitting.

## Required outputs

1. Lecture style theory coursebook.
2. Original practice workbook.
3. Subject issue atlas.
4. Thirty day starter plan.
5. Operating dashboard.

## Quality rules

- Use official SQE pages, SRA pages, public legislation, public procedure rules, and public case-law sources.
- Each topic should include rules, triggers, traps, client-fact application, and a brief self-test item.
- Avoid filler sections and repeated generic paragraphs.

## Section template

For each subject, write:

1. What is tested.
2. Core rule map.
3. Trigger words.
4. Standard client scenario.
5. Trap and exception.
6. Ethics overlay.
7. Self-test item and explanation.
